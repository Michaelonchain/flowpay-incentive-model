# FlowPay Incentive Model

This repository documents the incentive structure of the FlowPay (FPAY) token.

Goal: design a system where users are rewarded for behaviors that strengthen the protocol.

## Core Incentives

1. Liquidity Providers
Users who provide liquidity receive rewards in FPAY tokens.

2. Validators / Security
Participants who secure the network earn emissions.

3. Long-Term Holders
Users who lock tokens receive higher rewards.

4. Active Users
Users who utilize the protocol for payments or DeFi interactions may earn small rewards.

## Objective

The objective of this incentive model is to:

- encourage long-term participation
- reduce sell pressure
- reward useful activity
- strengthen network security

Good incentives create sustainable ecosystems.
